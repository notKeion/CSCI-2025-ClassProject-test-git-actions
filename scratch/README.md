# Scratchwork Folder  

This folder contains scratchwork and exploratory code for the project. It includes:  

- **R Scripts**: Temporary or experimental R code.  
- **Jupyter Notebooks**: Notebooks for data analysis, visualization, and prototyping.  

### Notes  
- Files in this folder are not part of the final deliverables.  
- Use this space for testing ideas and documenting intermediate steps.  

### Folder Structure  
- `R/<YOUR-NAME>`: Contains R scripts.  
- `notebooks/<YOUR-NAME>`: Contains Jupyter notebooks.

Keep the folder organized and clean up unused files periodically.

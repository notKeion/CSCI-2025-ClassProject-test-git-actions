# Data Directory
This dictionary contains the datasets used in this project. Each dataset is organized into subdirectories based on its type and purpose.

## Directory Structure
- `raw/`: Contains the original, unprocessed datasets.
- `processed/`: Contains datasets that have been cleaned and preprocessed for analysis.
- `external/`: Contains datasets obtained from external sources, if needed.

## Usage
When working with the datasets, ensure to reference the appropriate subdirectory based on whether you need raw or processed data. Always document any changes made to the datasets in the `processed/` directory for reproducibility, additionally, please include all changes in your pull requests.
